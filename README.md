## Prerequisites & Packages Used

- cv2
- mysql.connector
- pandas
- csv
- numpy
- tkinter, PIL
- os, datetime, time, 

Directories will be created during first runtime. 
On encountering database / SQL errors, open the .sql file in mysql and drop all the tables and recreate them.
